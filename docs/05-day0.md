# Day 0：环境准备（启动前一次性做完）

> **若主跟《Go 23 天学习计划 v2》**：Day 0 以 **v2 总览 docx** 为准。本页对应旧「3 周 / 20 天」线，可作参考。

---

正式开始前，把环境搭好。预计 **30–60 分钟**。

## 安装 Go

```bash
brew install go
go version
```

应看到：`go version go1.22+` darwin/amd64 或 arm64。

```bash
go env GOPATH GOROOT
```

## VS Code + Go 插件

- 安装 VS Code（若未装）  
- 安装 Go 插件：搜索 `golang.go`  
- 插件提示安装 `gopls`、`dlv` 等工具时，全部 Install

## 建项目目录

```bash
cd ~/Projects   # 或你习惯的目录
mkdir mini-search && cd mini-search
go mod init github.com/yourname/mini-search   # 换成你的 GitHub 用户名
mkdir -p cmd/server internal/handler internal/service internal/client
touch cmd/server/main.go README.md
```

（你当前仓库若已用别的模块路径或目录名，以你本仓库为准，不必强行改名。）

## 安装 Docker（Week 3 Kafka 用）

- Docker Desktop for Mac：`brew install --cask docker`  
- **现在不用装 Kafka**，Week 3 再用 docker-compose 启动

## Day 0 验收

- `go version` 能跑  
- VS Code 打开 `main.go` 有语法高亮  
- mini-search 目录结构建好  
- Docker 装了（不用启动）
